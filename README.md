# Savory Bites Restaurant Website

A responsive static website for Savory Bites Restaurant, showcasing the menu, gallery, reservations, and contact information.

## Features

- **Home Page**: Hero section, about us, and featured dishes
- **Menu Page**: Display of food and drink offerings
- **Gallery Page**: Photo gallery of the restaurant and dishes
- **Reservations Page**: Reservation form
- **Contact Page**: Contact information and contact form

## Tech Stack

- **HTML5**: Structure and content
- **CSS3**: Styling and responsive design
- **JavaScript**: Interactive elements and form handling

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/Kiru-6816/savory-bites-restaurant.git
   cd savory-bites-restaurant
   ```

2. Open `index.html` in your web browser to view the website.

## Usage

Navigate through the pages using the navigation menu. The website is fully responsive and works on desktop and mobile devices.
